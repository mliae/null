# 模板介绍

空值二开，进阶模板，适合自媒体用户。

# 关于使用
 - 将本主题里的所有文件放在您网站目录的content/templates内，注意文件夹名字必须为NullTwo

 - 后台->模板->启用本主题->进入模板设置配置信息

 - 本主题限个人使用，公开发布请注明原作者：二呆，及链接：http://www.tongleer.com
